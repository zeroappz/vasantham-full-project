Medical and Doctor Website CMS

Introduction

Vasantham Health Centre is a responsive Medical and Doctor Website Content Management System (CMS). It is a complete solution for a doctor or medical website. This CMS is built completely with PHP and MySQL. It has a nice and attractive front end and back end interface that are really awesome and eyecatching. The back end has a lot of nice and maintainable features that are needed for a modern and professional website to handle the contents easily by client. Almost all kinds of changes are possible to do using the admin panel without having any kind or programming language knowledge.

This CMS is built using the procedural php and without any framework. For this reason, it can be customized easily by most of the developers. This CMS script was prepared using the PDO feature of PHP language. That's why, no SQL injection and XSS attack is possible here and your website will be safe. Also in all pages, admin level security and invalid page url submission checking were done.

Installation

In order to install the CMS, please follow these steps:

  1. Search for the zip file cms.zip in the main theme folders that you have downloaded.
  2. Now upload the cms.zip file into your server using FTP or Cpanel. Extract this and use. You can use our cms into root domain or subdomain.
  3. Create mysql database using cpanel or your hosting providers' system. Then import the Vasantham Health Centre.sql file (that you got into the theme's folder) into your created database.
  4. Now you will have to edit the configuration file. Into our CMS folder, go to: admin > config.php and open it with a text editor. Change the following marked information:

Dashboard

In the dashboard, you will be able to get all the total data for news, categories, users, photos, videos and subscribers. From the top right corner menu, any user can change his profile information, password, photo etc.

User

In this CMS, unlimited users can be created under total 2 roles, one is admin and another is publisher. Here user's email address will be treated as username in this website. User photo upload is optional field. User can be setup as active, edit or delete.

Admin can do almost everything of this website, except removing Super Admin and managing the users.

Publisher can do only work with file, news, photo, video and subscriber options.

Settings

In this cms, you can change the following things in the settings:

  • Logo Image
  • Favicon Image
  • Footer About Us section, copyright section, contact or website email, number, fax and google map code
  • Contact form email address, subject and thank you message
  • Setting up how many recent news and popular news will be shown in the sidebar, footer and home page
  • Home page meta title, meta keywords and meta description
  • Home page title, subtitle and active or inactive status setup for service, department, doctor, pricing, testimonial, latest news and partner.

Page

When you will try to add a new page, you will have to follow these steps:

  1. In the main page, click on Add New.
  2. Put Page Name in the text field. It is mandatory.
  3. You can put page slug manually, or keep empty. If you keep this empty, slug will be automatically generated by the system.
  4. Place the Page Content into the content editor.
  5. Select your desired Page Layout from the drop down.
  6. Upload a Banner for the page to show in the front page.
  7. You can make the page Active or Inactive.
  8. Put the SEO information (meta title, meta keyword and meta description).

You can edit, delete, activate or deactivate page any time from the page main page.

Menu

You can add category item, page item, or any other manual link as the menu in this cms. When you will add category or page item as your menu item, select the desired category or page from the dropdown, then give them a order and select the parent for them. If you want to setup that menu as main menu item, select No Parent option from the dropdown, otherwise select another menu as parent. In Other Menu section, you can setup an external link or manual link and name as menu item. In this case, you will get an extra URL field to put url manually.

Advertisement (Home)

There are total 2 fixed locations to put your advertisement in the home page as static advertisement. One location is into the header and another location is under the featured news section. From the backend, go to Advertisement (Home) section. You will get there total two tabs: Inside Header and Under Featured News.

You can link the advertisement to an external URL or not. Put a jpg, jpeg, png or gif photo as advertisement. Recommended size of banner for advertisement (home) is given below:

  1. Inside Header: Recommended size is: 848X130.
  2. Under Featured News: Recommended size is: 1170X120. But you can use long height image here.

Advertisement (Sidebar)

There are total 2 locations to put your advertisement in sidebar. You can add unlimited photos in those sections. One section is Sidebar Top and another is Sidebar Bottom. From the backend, go to Advertisement (Sidebar) section. You can add, edit, delete and show sidebar advertisements.

You can link the advertisement to an external URL or not. Put a jpg, jpeg, png or gif photo as advertisement. Recommended width of banner for advertisement (sidebar) is 500px. You can make height as long as you can.

Advertisement (Home Category)

This section is a bit tricky. It can be seemed as tough, but it is easy. We are explaining the procedure below.

You know there are some categories in the home page. You can setup which categories you want to show on the home page. We already have showed it. Now, in this advertisement section, just add categories one by one and those will be shown on below the home page category sections following order.

From the backend, go to Advertisement (Category) section. You can add, edit, delete and show advertisements.

You can link the advertisement to an external URL or not. Put a jpg, jpeg, png or gif photo as advertisement. Recommended size for these advertisements is: 750X120. You can make height as long as you can.
